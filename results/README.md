# Research results

This directory contains curated aggregate outputs only. Raw market data,
point-in-time feature tables, target caches, databases, logs, and sealed
holdout observations are never committed.

`phase1_5m_discovery_through_20260430` is restricted to discovery data ending
April 30, 2026. May 1, 2026 onward was not accessed.

`phase1_final_discovery_through_20260430` contains the final corrected Phase 1
screen, exact diagnostics, descriptive regime/time/scope findings, and a
strategy-research interpretation. Its compressed machine-readable archive
contains the full broad-screen results without committing source data or caches.
